# Security Research Profile

## Overview

Firman Siregar is a smart contract security researcher focused on vulnerability discovery, DeFi protocol security, and AI-assisted auditing methodologies.

His research explores the intersection of blockchain security, adversarial thinking, and emerging AI capabilities to improve the efficiency and depth of smart contract security analysis.

---

## Areas of Research

Primary research interests include:

* Smart contract vulnerability discovery
* DeFi protocol attack surface analysis
* Flashloan-enabled exploit scenarios
* Protocol threat modeling
* Security auditing methodologies
* AI-assisted vulnerability detection

---

## Technical Focus

Security research within this repository focuses on the analysis of:

* EVM-based smart contracts
* DeFi protocols and financial primitives
* composable DeFi attack vectors
* protocol invariant violations
* accounting and state transition vulnerabilities

---

## Research Methodology

Security analysis typically follows a structured approach:

1. Protocol architecture review
2. Threat modeling
3. State transition analysis
4. Vulnerability pattern matching
5. Adversarial attack scenario construction
6. Validation and reporting

This structured approach is documented further in the repository's **methodologies/** directory.

---

## AI-Assisted Security Research

This repository also explores the use of AI tools to enhance smart contract security research.

Key areas include:

* prompt engineering for vulnerability discovery
* AI-assisted threat modeling
* automated vulnerability pattern detection
* AI-assisted audit workflow optimization

Prompt frameworks developed for this research are available in the **ai-audit-prompt-packs/** directory.
