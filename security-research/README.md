# Security Research

This section contains research articles and technical writeups related to smart contract security and decentralized finance (DeFi).

The focus is on studying vulnerability patterns, attack techniques, and common design flaws found in blockchain-based systems.

## Topics Covered

Examples of research topics include:

* Flashloan attack patterns
* Oracle manipulation attacks
* DeFi accounting vulnerabilities
* Reentrancy variants
* Protocol invariant violations
* ERC token standard security considerations

## Purpose

The goal of this research is to:

* Understand how real-world exploits occur
* Identify recurring vulnerability patterns
* Improve auditing methodologies
* contribute to the broader Web3 security ecosystem

Each article aims to translate technical vulnerabilities into clear security insights that can help developers and auditors build safer smart contract systems.
