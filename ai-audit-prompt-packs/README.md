# AI Audit Prompt Packs

This directory contains structured prompt frameworks designed to assist AI models in performing smart contract security analysis.

These prompt packs are part of an ongoing research effort exploring how large language models can augment traditional smart contract auditing workflows.

---

## Purpose

Smart contract auditing requires structured reasoning across multiple dimensions, including:

* protocol architecture
* state transition logic
* financial accounting mechanisms
* adversarial attack modeling

These prompt packs are designed to guide AI systems through a systematic security analysis process.

---

## Available Prompt Packs

### EVM Audit Prompt Pack

`ultra-premium-audit-prompt-pack-evm.md`

Designed for auditing EVM-based smart contracts, including DeFi protocols, token systems, and financial primitives.

---

### Move Audit Prompt Pack

`ultra-premium-audit-prompt-pack-move.md`

Adapted for Move-based ecosystems such as Sui and Aptos, focusing on resource-oriented state management and capability-based access control.

---

### Solana Audit Prompt Pack

`ultra-premium-audit-prompt-pack-solana.md`

Designed for Solana programs written in Rust, including account-based logic analysis and CPI interaction risks.

---

### NEAR Audit Prompt Pack

`ultra-premium-audit-prompt-pack-near.md`

Adapted for NEAR smart contract architecture and asynchronous execution patterns.

---

### Pre-Deployment Incident Likelihood Prompt Pack

`pre-deployment-incident-likelihood-prompt-pack.md`

A framework designed to simulate potential exploit scenarios before protocol deployment by modeling adversarial strategies and attack surfaces.

---

## Research Objective

This research explores how AI can assist security researchers in:

* identifying vulnerability patterns
* modeling exploit scenarios
* analyzing protocol invariants
* accelerating audit workflows

The goal is not to replace human auditors but to augment their analytical capabilities.

---

## Status

These prompt packs are part of ongoing experimentation in **AI-assisted smart contract security research** and may evolve over time as models and auditing techniques improve.
