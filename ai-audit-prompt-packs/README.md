# AI Audit Prompt Packs

This directory contains structured prompt packs designed to assist large language models (LLMs) in performing smart contract security analysis.

These prompt packs are part of my research into **AI-assisted auditing workflows** and aim to improve the efficiency and depth of vulnerability discovery across different blockchain ecosystems.

## Supported Ecosystems

The prompt packs currently cover multiple smart contract environments:

* EVM-based protocols
* NEAR Protocol
* Solana (Rust-based programs)
* Move-based ecosystems (e.g., Sui / Aptos)

## Purpose

These prompt frameworks are designed to guide AI systems through structured security analysis steps such as:

* protocol architecture understanding
* attack surface mapping
* invariant discovery
* vulnerability pattern detection
* exploit scenario modeling

## Research Goal

The objective of these prompt packs is to explore how structured prompts can augment traditional smart contract auditing and help researchers identify vulnerabilities more efficiently.

This work forms part of my broader research into **AI-assisted smart contract security analysis**.
